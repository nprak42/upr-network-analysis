About This Project

This project analyzes collaboration networks among civil society organizations (CSOs) participating in the UN Universal Periodic Review. By examining joint submissions, we can identify key organizations, brokerage positions, and patterns of advocacy coalition formation.

Each network visualization shows organizations as nodes and collaborations as edges, with size and color indicating centrality and brokerage positions. Click any country above to explore its network structure and key metrics.
